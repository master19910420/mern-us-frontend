# Business Experience Assessment

A React application for testing the business experience of candidates. Candidates register with their details, then complete a 6-question assessment with single-choice answers.

## Features

- **Sign up**: Full name, email, years of experience, social profile link, gender
- **Success notification**: Professional message after registration, then redirect to assessment
- **Assessment**: One question per page, 4–5 answers each, single choice; Next/Previous navigation

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Tech

- React 18
- React Router 6
- Vite 5
